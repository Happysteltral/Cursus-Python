# Python Cursus SJABI — Website

Welkom bij de Python Cursus SJABI!
Dit project bevat een educatieve website die leerlingen van SJABI begeleidt bij het aanleren van Python‑programmeren, van absolute beginner tot gevorderde basis.

# Over het project
Deze website is ontwikkeld om leerlingen te ondersteunen bij:

- het leren van de basis van Python
- het oefenen van programmeervaardigheden
- het uitvoeren van opdrachten en challenges
- het volgen van een gestructureerde leerlijn

De inhoud is afgestemd op het niveau van leerlingen in de derde graad secundair onderwijs.

#  Hoe de website gebruiken?

Open de website via index.html.
Navigeer door de lessen in de bovenste menubalk.
Maak de opdrachten in de juiste volgorde.
Check oplossingen of hints indien beschikbaar.
Experimenteer met de codevoorbeelden!

Voor de opdrachten wordt aangeraden om Thonny of VS Code te gebruiken.
