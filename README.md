# Cursus Python

A comprehensive Python programming course for students, built with MkDocs.

## About

This repository contains a complete Python course website designed to help students learn Python from the ground up. The course covers everything from basic concepts to advanced topics with interactive examples and exercises.

## Quick Start

### View the Course

Visit the live website: [https://happysteltral.github.io/Cursus-Python/](https://happysteltral.github.io/Cursus-Python/)

### Local Development

To run the documentation locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Happysteltral/Cursus-Python.git
   cd Cursus-Python
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Serve the documentation**
   ```bash
   mkdocs serve
   ```

4. **Open in browser**
   Navigate to `http://127.0.0.1:8000/`

### Build the Documentation

To build the static site:

```bash
mkdocs build
```

The generated site will be in the `site/` directory.
