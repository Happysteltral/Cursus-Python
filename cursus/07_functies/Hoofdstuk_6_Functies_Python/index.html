    • Wat een functie is

    • Hoe je een functie definieert
    • Wat parameters zijn
    • Wat een returnwaarde is
    • Het verschil tussen lokale en globale variabelen

    1️⃣ Waarom functies gebruiken?

    Stel dat je meerdere keren dezelfde berekening nodig hebt in je programma.\ Zonder functies moet je telkens dezelfde code opnieuw schrijven.

    Dat is:

    • ❌ Onoverzichtelijk\
    • ❌ Gevoelig voor fouten\
    • ❌ Moeilijk aan te passen

    Met functies kan je:

    • ✅ Code hergebruiken\
    • ✅ Je programma overzichtelijk maken\
    • ✅ Grote problemen opsplitsen in kleine stukjes

    !!! note Een functie is een stukje code met een naam dat je kan oproepen wanneer je het nodig hebt.

    2️⃣ Een eerste functie

    🔹 Structuur

    def naam_van_de_functie():
    # code die uitgevoerd wordt

    🔹 Voorbeeld

    def begroet():
    print("Hallo leerling!")

    Deze functie doet nog niets zolang we ze niet oproepen.

    🔹 Functie oproepen

    begroet()

    Volledig programma:

    def begroet():
    print("Hallo leerling!")

begroet()

    !!! example Wat wordt er geprint als je begroet() drie keer onder elkaar zet?

    3️⃣ Functies met parameters

    def begroet(naam):
    print("Hallo", naam)

    Gebruik:

    begroet("Nicky")
begroet("Sarah")

    !!! note Een parameter is een variabele die tijdelijk bestaat binnen de functie.

    4️⃣ Meerdere parameters

    def som(a, b):
    print(a + b)

som(3, 5)

    !!! tip De volgorde van de parameters is belangrijk!

    5️⃣ Returnwaarde

    def som(a, b):
    return a + b

    Gebruik:

    resultaat = som(4, 6)
print(resultaat)

    !!! important return geeft een waarde terug en stopt onmiddellijk de functie.

    6️⃣ Print vs Return

    ❌ Met print

    def som(a, b):
    print(a + b)

    ✅ Met return

    def som(a, b):
    return a + b

    7️⃣ Lokale en globale variabelen

    def test():
    x = 5
    print(x)

test()

    !!! warning Variabelen die in een functie gemaakt worden, bestaan alleen binnen die functie.
