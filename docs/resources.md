## Books

### Beginner
- **Python Crash Course** by Eric Matthes - Great for beginners
- **Automate the Boring Stuff with Python** by Al Sweigart - Practical Python projects
- **Learn Python the Hard Way** by Zed Shaw - Learning by doing

### Intermediate
- **Fluent Python** by Luciano Ramalho - Deep dive into Python
- **Effective Python** by Brett Slatkin - Best practices and tips
- **Python Tricks** by Dan Bader - Python features and idioms

### Advanced
- **Python Cookbook** by David Beazley and Brian K. Jones - Advanced recipes
- **High Performance Python** by Micha Gorelick and Ian Ozsvald - Optimization techniques
