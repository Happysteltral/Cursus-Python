# 15. Mini-project

## Klassenlijst beheren

Maak een programma waarmee je:

- namen toevoegt
- namen verwijdert
- alle namen toont
- controleert of een leerling aanwezig is

Voorbeeld:

```text
1. Naam toevoegen
2. Naam verwijderen
3. Toon lijst
4. Zoek leerling
```

!!! tip "Uitbreiding"
    Voeg een menu toe met een `while`-lus zodat het programma blijft werken tot de gebruiker stopt.
